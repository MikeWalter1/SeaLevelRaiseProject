Create a new Project

In order to be able to create projects, you have to have created an Organization first.

Rules:

  1. Only one project can be open at a time.
  2. Projects have to be about climate change.
  3. Projects have to be related to small-states.
  4. Projects have to be non-profit.
  5. The max amount of funds needed can't exceed x ETH.
  6. Once a project is created it has X weeks to be voted by the DAO community.
  7. Once a project is voted, the funds will be sent to the organization.
  8. After the project is finished, the organization has to provide a proof.
